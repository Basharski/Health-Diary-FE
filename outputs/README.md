# outputs – Robot Framework testitulokset

Tässä kansiossa on Robot Framework -testien tuottamat loki- ja raporttitiedostot.

Testit ajetaan komennolla:

```bash
robot --outputdir outputs tests/
```

## Viimeisin testiajo

| Tiedosto | Kuvaus |
|----------|--------|
| [report.html](report.html) | Yhteenveto testien suorituksesta |
| [log.html](log.html) | Yksityiskohtainen lokitiedosto testitapauksittain |
| [output.xml](output.xml) | Koneluettava XML-tulostiedosto |
